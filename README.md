- Neo Brutalist UI
- Interactive learning roadmaps
- Animated layouts
- Dynamic roadmap modals
- Curated tech resources
- Responsive design

# ✨ Features

## 🎯 Interactive Career Discovery

Explore multiple career paths including:

- 🎨 Frontend Wizard
- ⚙️ Backend Overlord
- 🧠 Fullstack Beast
- ✏️ UI/UX Architect
- 📊 Data Scientist
- ☁️ DevOps Ninja
- 📱 Mobile Creator
- 🧩 Product Manager

Each role contains:
- Salary insights
- Learning phases
- Skill progression
- Course recommendations
- Certifications
- Career roadmap

---

## 🧠 Dynamic Roadmap System

Every role opens an immersive animated roadmap modal featuring:

✅ Timeline progression  
✅ Phase-based learning  
✅ Recommended skills  
✅ FREE & PAID resources  
✅ Certification suggestions  
✅ Smooth animations
